深度数据分析报告：渠道交易结构特征 (v3)

摘要: 本报告基于 2025-12-25 至 2026-01-13 的 81,516 条交易记录，针对不同渠道在时间、商品、促销维度的交易特征进行客观统计分析。

1.2 指标定义 (Definitions)

  • 折扣率: 总折扣金额 / (总实收金额 + 总折扣金额)。
  • Promo Uplift: (促销单AOV - 正价单AOV) / 正价单AOV。正值表示促销订单的客单价高于正价订单，负值表示低于。

1.3 数据现象 (Observations)

  1. 促销结构差异: 万家App 的促销订单占比最高 (93.7%) 且折扣率最高 (19.5%)，但其 Promo Uplift 为负值 (-11.0%)，表明该渠道促销订单的平均金额低于非促销订单。
  2. 正向激励: 万家小程序 Promo Uplift 达到 +43.5%，且 UPT (5.0) 居所有渠道之首。数据表明该渠道促销订单往往伴随着更高的客单价和件数。
  3. 客单分层: 美团外卖、京东小时购、万家小程序的 AOV 均在 ¥56-60 区间，显著高于万家App 和饿了么的 ¥41-42 区间。
